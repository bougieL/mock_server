## Features
基于mockjs的数据模拟服务，同时支持jsonp。

## Installation
```bash
git clone https://github.com/bougieL/mock_server.git
cd mock_server
npm i
```

## API Documentation
### scripts:

```bash
#开启server并watch文件
npm run dev

#开启server
npm run server

#watch文件
npm run watch

#生成文档
npm run doc
```
